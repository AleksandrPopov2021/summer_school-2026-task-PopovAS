## Детальное описание экранов

### SCR-001: Splash Screen

**Название:** Экран загрузки приложения  
**Описание:** Начальный экран, отображаемый при запуске приложения. Показывает логотип скалодрома и индикатор загрузки.  
**Приоритет:** MVP  
**Связанные требования:** FR-026 (регистрация)  
**Источники:** domain-description.md, раздел 4.4

**Элементы экрана:**
- Логотип скалодрома «Вертикаль»
- Индикатор загрузки (spinner/progress bar)
- Версия приложения (опционально)

**Переходы:**
- → SCR-002 (если пользователь не авторизован)
- → SCR-003 (если пользователь авторизован)

### SCR-003: Schedule Screen

**Название:** Экран расписания тренировок  
**Описание:** Основной экран для просмотра доступных слотов тренировок на 7 дней вперёд с навигацией по дате и фильтрацией.  
**Приоритет:** MVP  
**Связанные требования:** FR-001, FR-002, FR-003, FR-004, FR-005, FR-006, FR-007, FR-008, FR-009  
**Источники:** UC-001, US-001, US-002, US-022, US-023, domain-description.md, раздел 4.1

**Элементы экрана:**
- Календарь/переключатель дат (7 дней вперёд)
- Список слотов тренировок на выбранную дату
- Для каждого слота:
  - Дата и время начала
  - Длительность (~1,5 часа)
  - Зона/формат тренировки (болдеринг, трассы с верёвкой)
  - Инструктор (ФИО, средняя оценка — Post-MVP)
  - Количество свободных мест (осталось X из Y)
  - Прокатный тариф
  - Адрес скалодрома
  - Кнопка записи (скрыта/заблокирована при 0 мест или < 30 мин до начала)
  - Пометка об отмене (для отменённых скалодромом слотов)
- Empty state: «Пока нет доступных тренировок» (если слотов нет)
- Индикатор загрузки

**Переходы:**
- → SCR-004 (при выборе слота)
- → SCR-013 (empty state при отсутствии слотов)

**Бизнес-правила:**
- BR-006: Запись на слот возможна не позднее чем за 30 минут до начала
- BR-007: На тренировки «трассы с верёвкой» запись только при допуске инструктора
- BR-008: При 0 свободных мест слот отображается без кнопки записи
- BR-009: Отображать конкретное число свободных мест в формате «осталось X из Y»
- BR-019: Отменённый скалодромом слот остаётся в списке с пометкой об отмене
- BR-027: По умолчанию слоты на 7 дней; навигация по дате
- BR-035: Средняя оценка инструктора видна клиенту до записи (Post-MVP)

### SCR-004: Slot Detail Screen

**Название:** Экран деталей слота тренировки  
**Описание:** Экран с подробной информацией о выбранном слоте тренировки перед записью.  
**Приоритет:** MVP  
**Связанные требования:** FR-003, FR-004, FR-007, FR-008, FR-009  
**Источники:** UC-001, UC-002, US-002, US-003, US-004, domain-description.md, раздел 4.1

**Элементы экрана:**
- Дата и время начала тренировки
- Длительность (~1,5 часа)
- Зона/формат тренировки (болдеринг с инструктажем / трассы с верёвкой)
- Информация об инструкторе:
  - ФИО
  - Средняя оценка (звёзды 1–5, Post-MVP)
- Количество свободных мест (осталось X из Y)
- Прокатный тариф
- Адрес скалодрома
- Статус доступности:
  - Доступен (кнопка «Записаться»)
  - Нет мест (сообщение «Мест нет», кнопка скрыта/заблокирована)
  - Менее 30 минут до начала (сообщение «Запись закрыта», кнопка заблокирована)
  - Требуется допуск (для опытных тренировок без допуска)

- Кнопка «Записаться» (если слот доступен)
- Кнопка «Назад»

**Переходы:**
- → SCR-005 (при нажатии «Записаться»)
- → SCR-003 (кнопка «Назад»)

**Бизнес-правила:**
- BR-006, BR-007, BR-008, BR-009, BR-019, BR-035

### SCR-005: Booking Screen

**Название:** Экран оформления записи на тренировку  
**Описание:** Экран выбора снаряжения, подтверждения согласия на риск (для первой записи) и оформления записи на тренировку.  
**Приоритет:** MVP  
**Связанные требования:** FR-010, FR-011, FR-012, FR-013, FR-014, FR-015  
**Источники:** UC-002, US-005, US-006, US-007, domain-description.md, раздел 4.5, раздел 4.6, раздел 5.1

**Элементы экрана:**
- Информация о слоте (дата, время, зона, инструктор)
- Выбор снаряжения:
  - Переключатель «Своё снаряжение» / «Прокат»
  - Чекбоксы для позиций проката:
    - Скальные туфли
    - Страховочная система
    - Каска
    - Магнезия
  - Возможность комбинации своё + прокат
- Разбивка стоимости:
  - Стоимость тренировки
  - Стоимость проката (по позициям)
  - Итого
- Чекбокс подтверждения согласия на риск (для первой записи)
- Кнопка «Подтвердить запись»
- Кнопка «Отмена»
- Индикатор загрузки при отправке запроса
- Сообщение об ошибке (при отказе бэкенда)
- Сообщение об успехе (при подтверждении записи)

**Переходы:**
- → SCR-015 (если первая запись, до подтверждения согласия)
- → SCR-006 (после успешной записи)
- → SCR-004 (кнопка «Отмена»)

**Бизнес-правила:**
- BR-002: Выбор снаряжения при записи; комбинация своё + прокат разрешена
- BR-003: Бронирование успешно только при подтверждении бэкендом
- BR-004: Двойная запись исключается на стороне бэкенда
- BR-031: При первой записи — подтверждение согласия на риск

### SCR-006: My Bookings Screen

**Название:** Экран списка активных записей клиента  
**Описание:** Экран для просмотра всех активных записей пользователя с возможностью отмены и перехода к деталям.  
**Приоритет:** MVP  
**Связанные требования:** FR-016, FR-017, FR-018, FR-019, FR-020, FR-021, FR-022, FR-023, FR-028  
**Источники:** UC-003, UC-008, UC-012, US-009, US-010, US-011, US-015, US-019, domain-description.md, раздел 4.4, раздел 5.2, раздел 5.3

**Элементы экрана:**
- Список активных записей (карточки)
- Для каждой записи:
  - Дата и время начала
  - Зона/формат тренировки
  - Инструктор
  - Статус оплаты (не оплачено, оплачено, возврат)
  - Кнопка «Отменить» (активна если > 1 часа до начала)
  - Кнопка «Изменить прокат» (если есть активная запись)
  - Пометка об отмене скалодромом (если применимо)
- Empty state: «У вас пока нет записей»
- Кнопка «Записаться на тренировку» (переход к расписанию)

**Переходы:**
- → SCR-007 (при выборе записи)
- → SCR-008 (при нажатии «Отменить»)
- → SCR-003 (кнопка «Записаться на тренировку»)

**Бизнес-правила:**
- BR-005: Клиент может иметь несколько активных записей одновременно
- BR-010: При отмене более чем за 2 часа — обычная отмена
- BR-011: При отмене за 1–2 часа — отмена с предупреждением
- BR-012: Менее чем за 1 час до начала отмена запрещена
- BR-016: При отмене скалодромом — статус «Отменена скалодромом» с причиной
- BR-025: Статусы оплаты: не оплачено, оплачено, возврат

## Глоссарий терминов экранов

| Термин | Определение |
|--------|-------------|
| Слот | Конкретное время тренировки с определёнными параметрами (зона, инструктор, количество мест) |
| Empty State | Состояние интерфейса при отсутствии данных для отображения |
| Push-уведомление | Системное уведомление на устройстве пользователя |
| Бейдж | Визуальный элемент, обозначающий достижение статуса |
| Прокат | Услуги предоставления снаряжения (туфли, система, каска, магнезия) |
